# Google PWA Checklist

<Appear>
  <p>Served over HTTPS</p>
  <p>Responsive design</p>
  <p>All app URLs available offline</p>
  <p>
    Manifest file for <em>Add to Home screen</em>
  </p>
  <p>Fast first load on "average" device</p>
  <p>Cross-browser support</p>
  <p>Non-blocking page transitions</p>
  <p>Each page has a URL</p>
</Appear>

# Google _Exemplary_ PWA Checklist

<Appear>
  <p>Site content indexed by Google</p>
  <p>Web ontology metadata (schema.org)</p>
  <p>Social metadata (OpenGraph)</p>
  <p>
    Canonical URL <code>&lt;link /></code> tags
  </p>
  <p>
    Uses browser <code>history</code> API
  </p>
  <p>No jumping content</p>
  <p>Scroll position retained on navigation</p>
  <p>Keyboard doesn't cover inputs</p>
  <p>Social share buttons</p>
  <p>Tasteful use of install banner</p>
  <p>Cache-first networking</p>
  <p>Offline prompt</p>
</Appear>
